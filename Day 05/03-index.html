Using predefined colors

1. Using RGB(Red, Green, Blue). These values ranges from 0-255.

2. Using RGBA(Red, Green, Blue, Alpha) where Alpha decides the transparency. Alpha values ranges from 0-1; 0 - lighter, 1 - darker

3. Using HEX

4. Using HSL(Hue, Saturation, Lightness, Transparency). Not recommended and not used that much.
